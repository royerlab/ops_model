          OPSin — OPS interpretability viewer
          OPSin is a phenotype hunter — it makes cell state visible to the human eye. In a pooled microscopy screen, switching off a gene reshapes cells in ways too subtle to see. A generative diffusion model, trained on millions of real cell images, renders a healthy cell to reveal each gene or complex's phenotype.
          The tabs
          Traversal — a healthy cell rendered to show the selected phenotype; the α slider sets the strength (5 = exaggerated).
          Embedding — every gene's model-rendered cell on a map, so similar effects sit together.
          Attn heads — where the model looked on the real cells (brighter = more informative).
          PCs — real cells sorted along the model's main axes of variation, with the biology each captures.
          Top cells — the real cells that show each phenotype most clearly.
